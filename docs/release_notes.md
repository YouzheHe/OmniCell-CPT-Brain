# Release Notes

This public code package was curated from the internal OmniCell_NVU working tree. The release keeps source code, lightweight configuration, and documentation while excluding large raw data, checkpoints, temporary backups, scheduler logs, generated figures, and intermediate result folders.

Private server paths were replaced with environment-variable placeholders. Some scripts still require project-specific data tables; those dependencies are documented in script arguments and in `docs/data_and_checkpoint_layout.md`.
